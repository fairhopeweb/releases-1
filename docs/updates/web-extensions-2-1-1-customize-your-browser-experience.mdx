The Pieces Web Extensions are already super useful — automatic related links, anyone?— but what would make them even better? A settings menu. With today’s release, you can customize your browser experience more than ever before with settings for the Pieces Web Extension!
## Settings Menu
In the settings menu, you can:
- Choose the position of Pieces buttons on code blocks
- Set the minimum number of characters required for the Pieces buttons to detect a code block
- Add websites to your Filter Out list, so that the Pieces buttons won’t appear on websites you do not want them to
![](rc="https://uploads-ssl.webflow.com/6143afec68f555387049efb3/630d88ec46bb6589a1a5bdb0_ezgif_com-gif-maker64_26dcd8121036d05955136a7251d5c74a_800.png)
To access the settings menu, click the Pieces logo in your browser and select “Settings.” The menu will pop up in a new tab in your browser.
![](uploads-ssl.webflow.com/6143afec68f555387049efb3/630d88ec46bb65428da5bdf0_ezgif_com-gif-maker65_217beab8d70ac04d191815293218f589_800.png)
Coming soon: the ability to choose how which buttons you want to appear on code blocks, the ability to toggle between light mode and dark mode, and more!
## Bug Fixes and Performance Improvements
This release also includes minor bug fixes and performance improvements, including improved code block detection and support for even more websites.
