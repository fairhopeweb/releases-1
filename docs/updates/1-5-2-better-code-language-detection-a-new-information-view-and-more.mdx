## Pieces for Developers
<em>Pieces for Developers is the core user-facing Pieces application. Get started for free <a href="http://code.pieces.app/install" target="_blank" rel="noopener noreferrer">here</a>.</em>
### Adding Pieces
- Language classification - Pieces auto-classifies your snippets by language using machine learning models. In this release, we pushed a couple major upgrades:<br/>Improved code language classification predictions<br/>We now display the confidence level on our code language classification
- Better image error handling - you’ll now receive an error message when trying to upload an unsupported image type
- Image size constraints - images uploaded to the app must be under 1MB
